# Q3 LazyColumn Student List

## Description
This project implements a scrollable list of student names using Jetpack Compose.  
The list is built using a `LazyColumn` and the `items()` DSL to efficiently display 20 student names.

I first created a basic version of the `StudentList` composable that focuses purely on functionality. That version is commented out in the code. I then created an updated version (the active, uncommented one) that includes additional UI elements such as padding, larger text, and borders to make the list easier to see and interact with during testing.

The original version is kept in the code for clarity and comparison.
